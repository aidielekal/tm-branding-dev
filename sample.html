Setting Up

  1. Font File: 'HK Grotesk Wide' and 'tmicon' are included in /css/fonts folder. Other fonts (Roboto & DM Sans) are accessible using Google Fonts. This file is important to ensure elements are working.
  2. Assets File: Included in /assets folder. Might be incomplete
  3. The only CSS file you needed is /css/tmbrand2022.css
  4. Use Bootstrap 5.
  5. Replace any code that linked to your directory (linked css,js,font files etc)

Buttons

Primary Button

Used to indicate primary call-to-action, with progressive arrow in its element.

  • Use btn-primary
  • Applying btn-sm will shrink the size to smaller size - may be used in mobile,
  • btn-sm can be used all across buttons variations

/** Button Primary **/
<button href="" class="btn btn-primary">
  <span class="text align-self-center"> Primary Button </span>
  <i class="icon-arrow-rightlg"></i>
</button>
/** for disabled **/
<button href="" class="btn btn-primary disabled">
  <span class="text align-self-center"> Primary Button </span>
  <i class="icon-arrow-rightlg"></i>
</button>
/** Small variant, btn-sm **/
<button href="" class="btn btn-primary btn-sm">
  <span class="text align-self-center"> Primary Button </span>
  <i class="icon-arrow-rightlg"></i>
</button>

For primary call-to-action with icons; in this sample, we are using fa-icons

  • Add class="icon" on i tags before fa-icon class
  • To differentiate on icon in primary buttons (those are enlarged icon)

/** Primary Button with icons **/
<button href="" class="btn btn-primary">
  <span class="text align-self-center"> Primary + icons </span>
  <i class="icon fa-solid fa-house"></i>
</button>

Secondary Button

Used to indicate secondary call-to-action,

  • Use btn-secondary
  • Applying btn-sm will shrink the size to smaller size - may be used in mobile,
  • btn-sm can be used all across buttons variations
  • You can change i elements to another secondary + icons combination, we are using fa-icon in this sample
  • You may remove i elements to create an alternate version of secondary button

/** Secondary button **/
<button href="" class="btn btn-secondary">
  <span class="text align-self-center"> Secondary </span>
  <i class="icon-arrow-rightlg"></i>
</button>
/** Secondary + icon **/
<button href="" class="btn btn-secondary">
  <span class="text align-self-center"> Secondary </span>
  <i class="icon fa-solid fa-house"></i>
</button>
/** Secondary alternate **/
<button href="" class="btn btn-secondary">
  <span class="text align-self-center"> Secondary </span>
</button>

Alternate Button

A simpler variation, and for alternative options

  • Use btn-alt
  • You can change i elements to another secondary + icons combination, we are using fa-icon in this sample

/** Alternative button **/
<button href="" class="btn btn-alt">
  <span class="text align-self-center"> Alternate </span>
  <i class="icon-arrow-rightlg"></i>
</button>
/** with fa-icons combination **/
<button href="" class="btn btn-alt">
  <span class="text align-self-center"> Search </span>
  <i class="fa-solid fa-magnifying-glass"></i>
</button>

Danger Button

For high impact/ danger action e.g. delete, cancel, and abort.
Use btn-danger class.

/** Danger button **/
<button href="" class="btn btn-danger">
  <span class="text align-self-center"> Delete </span>
</button>

Underline CTA and Link

For minor action. The a element are following the default style.
Use class underline-cta for a variant style of button.

Link: johndoe@email.com
Underline CTA: Clear filter
/** Underline CTA **/
<a href="" class="underline-cta">Clear filter </a>

a .underline-cta:hover, .underline-cta:hover{
  color:#757575;
}

Forms

Text Input

TM Branding input forms are very intuitive.
To ensure the elements are in place, try to stay close to example shown below,

  • Use the <form> tag
    • for form group, apply class="form form-group".
  • Make sure <input> is declared first before <label>.
  • The placeholder properties are only visible during ::focus state.
  • Label are required.

/** input type="text" **/
<div class="form-label-group">
  <input type="text" class="form-control form-text required" name="full_name" placeholder="Placeholder">
  <label for="fullname" class="form-label">Label</label>
</div>
